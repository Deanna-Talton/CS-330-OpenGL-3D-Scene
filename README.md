# CS-330-OpenGL-3D-Scene

This repository contains a portfolio artifact from CS 330: Computational Graphics and Visualization at Southern New Hampshire University. The included project demonstrates my ability to design and implement a fully realized 3D scene using OpenGL, applying concepts such as transformations, lighting, texturing, and camera controls.

The artifact focuses on building a detailed and interactive 3D environment by combining simple geometric shapes into complex objects. The project emphasizes scene composition, realistic lighting techniques, and user-controlled navigation to create an immersive visual experience.

Skills Demonstrated: OpenGL Rendering • 3D Transformations • Lighting (Ambient, Diffuse, Specular) • Texture Mapping • Camera Controls • Scene Composition • Modular Programming

---

## Reflection Journal

This project helped me shift my mindset from simply making something functional to intentionally designing a complete visual experience. I approached the design by breaking my scene down into smaller components and building each object from basic shapes, then refining them with textures and lighting. I followed an iterative process, starting with a simple layout and gradually adding complexity, which allowed me to make adjustments and improvements throughout development. This approach made the project more manageable and is something I can apply to future work.

While developing the program, I focused on organizing my code into separate functions for different responsibilities, such as object creation, transformations, and lighting setup. This made the code easier to read and debug as the scene became more complex. Iteration played a major role in my workflow, as I continuously tested changes to positioning, lighting intensity, and material properties to achieve the desired visual outcome. Over time, I became more deliberate in planning changes before implementing them rather than relying on trial and error.

This project demonstrated how computational graphics can combine creativity with technical problem-solving. I gained hands-on experience with 3D transformations, lighting models, and scene management, which are valuable skills for areas such as game development and simulation. These skills will support my future coursework and align with my long-term goal of working in a software or graphics-related field.
